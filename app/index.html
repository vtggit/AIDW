Data Ownership

AICRM is a backend-owned application. All business domains are stored through the backend API and PostgreSQL. Browser storage is used only for transient session or UI state.

  • Contacts: Backend-managed (PostgreSQL)
  • Templates: Backend-managed (PostgreSQL)
  • Leads: Backend-managed (PostgreSQL)
  • Activities: Backend-managed (PostgreSQL)
  • Settings: Backend-managed (PostgreSQL)

Settings Backup

Export or import Settings as a JSON snapshot. This is a client-side convenience tool for settings only — all business data (Contacts, Templates, Leads, Activities) is managed by the backend and is not affected.
